# AI Forest Canopy Segmentation

Deep learning project for forest canopy segmentation using TensorFlow, MobileNetV3, and U-Net.

## Features

- Forest canopy segmentation
- TensorFlow/Keras implementation
- MobileNetV3 encoder
- Dataset preprocessing
- Training and inference scripts
- TFLite export for deployment

## Project Structure

```
src/
├── preprocessing/
├── utils/
├── visualization/
├── train.py
├── predict.py
└── export_tflite.py
```

## Requirements

```bash
pip install -r requirements.txt
```
